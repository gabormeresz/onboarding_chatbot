# System Access Matrix

## Default Access by Role

### Software Engineer

- Email: Yes
- Microsoft Teams: Yes
- Git Repository: Upon request
- Jira: Upon request
- Production Systems: No

### Data Scientist

- Email: Yes
- Microsoft Teams: Yes
- Git Repository: Upon request
- Jira: Yes
- Analytics Platform: Upon request

### Manager

- Email: Yes
- Microsoft Teams: Yes
- Jira: Yes
- HR Reporting Tools: Yes

## Access Request Rules

- All non-default access must be requested via the IT Support Portal.
- Manager approval is required for:
  - Git access
  - Jira access
  - Analytics platforms
- Production system access requires:
  - Business justification
  - Security training completion
  - IT Security approval
