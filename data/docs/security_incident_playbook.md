# Security Incident Playbook

## What Is a Security Incident?

A security incident includes, but is not limited to:

- Suspicious VPN behavior
- Lost or stolen company devices
- Suspected phishing emails
- Unauthorized access attempts

## Immediate Actions

If a security incident is suspected:

1. Disconnect from the network
2. Do not attempt to investigate independently
3. Report the incident immediately to IT Security

## Reporting a Security Incident

- Security incidents must be reported via a high-priority IT ticket.
- Include as much detail as possible.
- IT Security will take ownership of the incident.
