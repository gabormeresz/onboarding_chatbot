# IT Troubleshooting Guide

## VPN Issues

### Common VPN Problems

- Incorrect password
- Expired password
- Network connectivity issues

### Self-Service Steps

1. Check internet connection
2. Restart the VPN client
3. Verify credentials
4. Reboot the device

If the VPN issue is **not resolved within 15 minutes**, contact IT Support.

## Email Issues

- If you cannot access email, verify that your password is not expired.
- Webmail access may work even if Outlook does not.

## Device Performance Issues

- Restart the device at least once per day.
- Ensure no unauthorized software is running.

## When to Contact IT Support

- Repeated errors
- Error messages you do not understand
- Any issue blocking your work
