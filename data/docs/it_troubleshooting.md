# IT Troubleshooting Guide

This guide provides basic self-service troubleshooting steps for common IT issues. Employees are encouraged to follow these steps before contacting IT Support to ensure quick resolution.

---

## VPN Issues

### Common VPN Problems

- Incorrect or mistyped password
- Expired password
- Multi-Factor Authentication (MFA) failure
- Unstable or restricted network connection
- Outdated VPN client version

---

### Self-Service Steps

Follow these steps in order:

1. Verify your internet connection by opening a public website
2. Confirm that your password is not expired
3. Ensure MFA device or app is available and working
4. Restart the VPN client
5. Disconnect and reconnect the VPN
6. Reboot the device
7. Try connecting from a different network (if possible)

If the VPN issue is **not resolved within 15 minutes**, contact **IT Support**.

---

## Email Issues

### Common Email Problems

- Unable to log in
- Password expired
- Outlook not syncing
- Emails not sending or receiving

### Self-Service Steps

- Verify that your password is not expired
- Try accessing email via **webmail**
- Restart the email client (e.g. Outlook)
- Restart the device
- Check internet and VPN connection

> ℹ️ Webmail access may work even if the desktop email client does not.

---

## Device Performance Issues

### Common Causes

- Long uptime without restart
- Too many background applications
- Unauthorized or resource-heavy software
- Low disk space

### Self-Service Steps

- Restart the device at least **once per day**
- Close unused applications and browser tabs
- Check available disk space
- Ensure no unauthorized software is running
- Run system updates when prompted

---

## Application Issues

- If an internal application is not working:
  - Check VPN connectivity
  - Verify system access rights
  - Restart the application
- If a web-based system fails:
  - Try a different browser
  - Clear browser cache
  - Check company status page (if available)

---

## Peripheral Issues

### Monitor / Keyboard / Mouse

- Check cable connections
- Reconnect USB devices
- Try a different USB port
- Restart the device

### Headset / Audio Issues

- Verify correct audio input/output device is selected
- Test in system audio settings
- Restart the application using audio (Teams, Zoom, etc.)

---

## When to Contact IT Support

Contact IT Support immediately if you experience:

- Repeated or recurring errors
- Error messages you do not understand
- Any issue blocking your work for more than **15 minutes**
- Security-related concerns (e.g. suspicious pop-ups, phishing emails)
- Lost or stolen devices

---

## Information to Provide to IT Support

When contacting IT Support, include:

- Description of the issue
- Error messages or screenshots (if available)
- Time when the issue started
- Steps already attempted
- Device type and operating system
